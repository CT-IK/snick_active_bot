# Фронтенд

Одностраничное приложение (SPA) на чистом JavaScript — **без сборки,
фреймворков и зависимостей**. Каталог [static/](../static/), отдаётся
FastAPI как статика; корневой `/` возвращает `index.html`.

## Файлы

| Файл | Назначение |
|------|-----------|
| `index.html` | Разметка: навбар, форма входа, вкладки, контейнер модального окна |
| `app.js` | Вся логика: авторизация, загрузка данных, рендеринг, модалки |
| `styles.css` | Стили (CSS-переменные для темы) |
| `hsm_logo.svg` | Логотип |

## Авторизация

- При загрузке `checkAuth()` берёт токен из `localStorage` (ключ `authToken`).
- Есть токен → `GET /api/users/me`, при успехе показывается приложение.
- Форма входа → `POST /api/auth/login`, токен сохраняется в `localStorage`.
- `getAuthHeaders()` формирует заголовок `Authorization: Bearer` для каждого
  запроса; `localStorage` — источник правды для токена.
- Ответ `401` → токен удаляется, показывается форма входа.

## Вкладки

`index.html` содержит 5 вкладок, переключение — `switchTab()`. Активная
вкладка определяет, что грузит `loadData()`:

| Вкладка | Источник данных |
|---------|-----------------|
| Задачи | `GET /api/tasks/` → `renderTasks()` |
| Таймлайн | `GET /api/tasks/` + `/api/workgroups/` → `renderTimeline()` |
| Рабочие группы | `GET /api/workgroups/` → `renderWorkgroups()` |
| Пользователи | `GET /api/users/` → `renderUsers()` |
| Шоб не забыть | `localStorage` → `renderStickyNotes()` |

## Таймлайн

`renderTimeline()` рисует для каждой задачи линию «В работе / На проверке»
с точками. Точки — это ответы на опросы (`poll_responses` задачи).
Позиция точки вычисляется в `getDotPositionPercent()` по индексу ответа и
их общему числу; выполненная задача (`done`) ставит последнюю точку в конец.

## Заметки «Шоб не забыть»

Заметки хранятся **только в `localStorage`** браузера (ключ
`tasktracker_notes`) — на сервер не отправляются и между устройствами
не синхронизируются.

## Модальные окна

Создание и редактирование сущностей — через одно модальное окно.
`showModal(title, html)` подставляет содержимое, `closeModal()` закрывает.
Формы строятся в функциях `showCreateTaskModal`, `editTask`,
`showCreateWorkgroupModal`, `editWorkgroup`, `showCreateUserModal`,
`editUser`. Закрытие — клик по фону или `Escape`.

## Автообновление

Каждые 25 секунд (`REFRESH_INTERVAL_MS`) активная вкладка
перезагружается, если пользователь авторизован.

## Утилиты

`formatDate`, `getStatusText`, `getStatusColor`, `getRoleText` —
форматирование дат, статусов и ролей для отображения. `escapeHtml` —
экранирование текста заметок.
